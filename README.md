# Thala-Aware 🩺

Thala-Aware adalah aplikasi *mobile* yang dirancang untuk meningkatkan kesadaran dan deteksi dini terhadap risiko Thalassemia. Aplikasi ini dikembangkan sebagai langkah nyata untuk mendukung pencapaian **Sustainable Development Goal (SDG) 3: Good Health and Well-being**, dengan memfasilitasi edukasi, pencatatan hasil skrining, serta menyediakan ruang diskusi interaktif.

## ✨ Fitur Utama
* **Autentikasi Aman:** Sistem Login dan Registrasi dengan manajemen sesi (*Session Management*).
* **Kalkulator Risiko & Jurnal Skrining:** Mencatat status pengguna, menghitung tingkat risiko, serta menyimpan log riwayat skrining lengkap (tanggal, lokasi, dan catatan).
* **Forum Diskusi Interaktif:** Ruang berbagi cerita dengan fitur *Create, Read, Update, Delete* (CRUD), *Like* secara *real-time* (Optimistic UI), dan sistem komentar.

## 🛠️ Teknologi yang Digunakan
* **Frontend:** Flutter & Dart
* **Backend:** PHP (Native REST API)
* **Database:** MySQL
* **Arsitektur:** MVC Sederhana (Models, Helpers/Services, Screens)
